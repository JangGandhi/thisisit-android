# thisisIT

IT 관련 소식과 정보를 제공하는 기사 애플리케이션으로, 수집형 요소를 도입하여 사용자가 자발적으로 학습하도록 설계했습니다. 본 프로젝트는 소프트웨어공학 과제로 제작되었습니다.

## 주요 기능

- 클릭하여 열람할 수 있는 다양한 기사 제공
- 기사를 사용자의 배경 지식 수준에 맞춰 두 단계(@everyone, @nerds)로 제공
- 사용자가 읽은 기사를 바탕으로 문제 출제
- 수집형 요소(SD카드) 도입
- 프로필에서 통계 출력
